DevTrack OS

DevTrack is a minimalist workspace designed for developers who value speed, structure, and professional growth. We automate the overhead so you can focus on building.

Focus First

Clean interfaces designed to reduce cognitive load and keep you in the flow.

Data Driven

Real-time analytics that track your productivity across all repositories.

GitHub Synchronization

DevTrack connects directly to your GitHub account to provide a seamless transition between coding and documenting.

  • OAuth-based secure authentication.
  • Live repository discovery.
  • Automated markdown publishing to specific repos.

Task Management

A streamlined engine for managing features, bugs, and learning goals. No complex overhead, just progress.

Tasks move through a simple, three-stage lifecycle: Backlog, In Progress, and Done. Once a task is completed, DevTrack can automatically commit the change to your repository.

Daily Logs

Documenting your journey is critical. Our Daily Log system is a Markdown-first editor with real-time preview and automated activity importing.

The "Magic" button pulls in your daily commits and completed tasks, generating a professional work summary in seconds.

Analytics

Visualize your output with premium heatmaps and streak counters. DevTrack monitors your commit frequency and task completion rates to provide deep insights into your developer velocity.

System Architecture

DevTrack is built with a modular Blueprint architecture, ensuring scalability and clean separation of concerns. The system is divided into functional modules that handle specific domains of the application.

Route Layer

Handles HTTP requests and serves as the API gateway for frontend interactions.

Service Layer

Contains the core business logic, including GitHub API integration and encryption.

We use Flask-SQLAlchemy for database interactions and Flask-Login for secure session management. Real-time features are powered by Socket.IO.

Technology Stack

The platform leverages a modern, lightweight stack designed for maximum performance and developer experience.

Backend

  • Python / Flask
  • SQLAlchemy (SQLite/PostgreSQL)
  • Flask-SocketIO
  • Flask-Migrate

Frontend

  • Jinja2 Templates
  • Tailwind CSS (CDN)
  • Vanilla JavaScript (ES6+)
  • Marked.js (MD Parsing)

Local Development

To run DevTrack locally, follow these steps to set up your environment:

# Clone the repository
git clone https://github.com/pandeji005/DevTrack.git

# Install dependencies
pip install -r requirements.txt

# Configure Environment
cp .env.example .env
# Edit .env with your GitHub Client ID/Secret

# Run the application
python run.py
