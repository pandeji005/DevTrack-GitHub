GitHub Synchronization

DevTrack connects directly to your GitHub account to provide a seamless transition between coding and documenting. 

# Example Sync Command
$ devtrack sync --repo=pandeji005/DevTrack

Local Setup

To run DevTrack locally for development:

  1. Clone the repository
  2. Install dependencies: pip install -r requirements.txt
  3. Configure .env with your GitHub API keys
  4. Run python run.py
